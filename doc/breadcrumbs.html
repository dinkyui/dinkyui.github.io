# Uso

Nesta seção aprenderemos a usar Breadcrumbs que são links de volta para cada página anterior pela qual o usuário navegou.

Inserindo breadcrumbs

O uso de breadcrumbs é um dos mais simples de todos os componentes que você já viu, veja abaixo um exemplo:

  • Home
  • Documentação
  • Breadcrumbs

<ul class="breadcrumb">
  <li><a href="#">Home</a></li>
  <li><a href="#">Documentação</a></li>
  <li>Breadcrumbs</li>
</ul>

Para alterar a cor de fundo do breadcrumbs basta adicionar o nome de uma cor como classe do elemento, veja um exemplo abaixo:

  • Home
  • Documentação
  • Breadcrumbs

<ul class="breadcrumb orange-darken">
  <li><a href="#">Home</a></li>
  <li><a href="#">Documentação</a></li>
  <li>Breadcrumbs</li>
</ul>
