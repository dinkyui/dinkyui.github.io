# Uso

Nesta seção aprenderemos a usar as cores (ver Paleta de cores abaixo) no texto e no fundo.

Cor de fundo

Para aplicar uma cor de fundo, basta adicionar o nome da cor como uma classe do elemento.
Cor de fundo blue-light

<div class="card blue-light">Cor de fundo blue-light</div>

Cor do texto

Para aplicar uma cor no texto, deve-se adicionar t- antes do nome da cor como classe do elemento.
Cor do texto t-blue

<div class="card t-blue">Cor do texto t-blue</div>
