# Uso

Nesta seção apresentaremos como usar os badges que são rótulos de notificação

Aplicando badges

Para aplicar badges, basta adicionar um elemento span com a classe badge tendo como conteúdo que irá ser exibido. Veja um exemplo abaixo: 


  <button class="btn blue">
    Meu botão
    <span class="badge">4</span>
  </button>

Alterando a cor do badge

Para alterar a cor do badge, basta adicionar o nome da cor como classe do elemento span. Veja: 


  <button class="btn red">
    Meu botão
    <span class="badge blue">4</span>
  </button>
