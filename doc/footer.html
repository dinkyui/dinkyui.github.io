# Uso

Nesta seção aprenderemos como utilizar footer

Footer simples

Este footer consiste apenas em um texto centralizado com uma linha superior

© 2019 My Website


<footer class="simple-footer">
  <p>© 2019 <span class="t-red-darken">My Website</span></p>
</footer>
