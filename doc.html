# Estrutura do projeto

Após realizar o download do arquivo .zip e descompactá-lo, os arquivos úteis estão na pasta css/. Esta pasta possui dois arquivos .css, um arquivo mimificado (.min.css) que aconselhamos o seu uso em produção (ou desenvolvimento) e outro arquivo que pode ser usado caso deseje modificar algo no arquivo (mais fácil para a compreensão). Importe qualquer arquivo para o seu projeto, neste caso usaremos o arquivo mimificado como exmplo. Organize a estrutura do seu projeto da seguinte forma: 


    meu-projeto/
    |----css/
    |    └--- dinky.min.css
    |
    └----index.html

# Estrutura do arquivo HTML

Abaixo é apresentado a estrutura de um arquivo HTML 


  

<html lang="pt-br">
<head>
  <meta charset="UTF-8">

  <!--Deixe navegador saber que o site é otimizado para celular-->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <!-- Titulo do seu site -->
  <title>meu-projeto</title>

  <!-- import do dinkyui.min.css -->
  <link rel="stylesheet" href="css/dinkyui.min.css">
</head>
<body>

  <!-- aqui fica o navbar -->
  
  <div class="container">
    <div class="row">
      <div class="col-12 sm-12 md-12 lg-12">
        <h1>Hello World</h1>
      </div>
    </div>
  </div>

  <!-- aqui fica o footer -->

  <!-- aqui fica os JS -->

</body>
</html>
